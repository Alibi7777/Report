# SAP ABAP REST API for Investment Report

This repository contains an SAP ABAP REST API implementation for retrieving investment report data from SAP BW/BPC and returning the result in JSON format.

The API is designed to be used by external systems, dashboards, or frontend applications that need to receive investment-related reporting data from SAP.
